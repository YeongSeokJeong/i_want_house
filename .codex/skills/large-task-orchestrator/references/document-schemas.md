# Document Schemas Index

Use these schema references for task-scoped continuity files in `./docs/orchestration/<task-name>/`:

- `plan.md` -> [plan-schema.md](plan-schema.md)
- `progress.md` -> [progress-schema.md](progress-schema.md)
- `decision.md` -> [decision-schema.md](decision-schema.md)
- `architecture.md` -> [architecture-schema.md](architecture-schema.md)

Backlog integration:

- Large task repo-level work/result status lives in `docs/backlog.md`, managed by `backlog-management`.
- Task-scoped orchestration files only link related `BL-*` IDs; they do not replace `docs/backlog.md`.
