# Wiki 구조 안내

이 위키는 JeonseLoop의 지속적인 설계, 운영 규칙, 개발 자동화 절차를 저장한다.

## 폴더 구조
- `domains/` - 업무 도메인별 지식 문서
- `decisions/` - 중요한 사람 의사결정 기록
- `rules/common/` - 공통 규칙과 원칙
- `rules/workflow/` - 반복 절차와 체크리스트

## 주요 진입점
- [[index]] - 전체 문서 목록
- [[SCHEMA]] - 작성 규칙과 라우팅 기준
- [[decisions]] - 의사결정 목록

## 작성 원칙
- 문서는 업무 개념 기준으로 분류한다.
- 파일명은 lowercase kebab-case를 사용한다.
- 위키 내용은 간결한 한국어로 작성한다.
